Ответы на вопросы

  1. Что представляют собой формы?

    Ответ: Форма — это компонент веб-страницы с элементами управления, такими как текстовые поля, кнопки, флажки, диапазон или поле выбора цвета.

  2. Для чего служат формы?

    Ответ: Пользователь может взаимодействовать с формой, предоставляя данные, которые затем могут быть отправлены на сервер для дальнейшей обработки (например, возвращая результаты поиска или вычислений).

  3. Каким типом задаётся поле ввода в форме текста?

    Ответ: input type="text" name="name" placeholder="Имя"

  4. Какими способами можно обратиться к элементу формы?

    Ответ:
    document.forms.form1.elements
    document.forms.elements['form1']
    window.document.forms.form1.elements

  5. Какой JS-код можно указать в атрибуте action?

    Ответ: Kод создания ссылки на нужную страницу или вызова функции отправки данных из формы. В сочетании с методом POST.

  6. Какой тип имеет свойство elements объекта Form?

    Ответ: Массив.

    Ответ: C помощью обработчика событий submit, помощью кнопки Enter

  11. Какие есть недостатки у стандартного способа задания валидации через HTML5?

    Ответ:

    • Применяются стандартные стили, которые нельзя изменить.
    • Эти стили применяются еще до того, как пользователь начнет работать с формой.
    • С его помощью можно проверить только факт заполнения полей, но не корректность введенных данных.
